Overview

PbPDFlib is a PureBasic-native PDF manipulation library that can create, read, and modify PDF files without installing Adobe Acrobat or any third-party dependencies.

The library is ported from the Go-language pdfcpu project and the C-language libharu project. It primarily follows libharu's API design style while incorporating pdfcpu's advanced features. All algorithms are implemented natively in PureBasic with no third-party dependencies.

Note:Before using the PbPDFlib library, add XIncludeFile "PbPDFlib.pb" at the beginning of your code to include the library.

Key Features

  • Create standards-compliant PDF documents, supporting PDF 1.2-2.0
  • Read and parse existing PDF files
  • Text drawing (Standard fonts + TrueType fonts + UTF-8 encoding)
  • Graphics drawing (Path/Stroke/Fill/Color/Transform matrix)
  • JPEG and PNG image loading and drawing
  • TrueType font subsetting
  • RC4-40/RC4-128/AES-128/AES-256 encryption
  • Bookmark/Outline creation and modification
  • Text, link, and highlight annotations
  • Text watermarks (rotation/opacity/diagonal)
  • Headers and footers
  • PDF merge/split/extract/delete pages
  • Extended graphics state (transparency/blend modes)

Example Code

Example 1: Create a Basic PDF Document

; PbPDFlib - Create a basic PDF document XIncludeFile "PbPDFlib.pb" ; Create document *doc.PbPDF_Doc = PbPDF_New() ; Add A4 page *page = PbPDF_AddPage(*doc) PbPDF_Page_SetPredefinedSize(*page, #PbPDF_PAGE_SIZE_A4, #PbPDF_PAGE_PORTRAIT) ; Write text PbPDF_Page_BeginText(*page) PbPDF_Page_SetFontAndSize(*page, "Helvetica", 24) PbPDF_Page_MoveTextPos(*page, 50, 750) PbPDF_Page_ShowText(*page, "Hello PbPDFlib!") PbPDF_Page_EndText(*page) ; Save file PbPDF_SaveToFile(*doc, "example1.pdf") PbPDF_Free(*doc)

Example 2: Drawing Graphics

; PbPDFlib - Drawing graphics XIncludeFile "PbPDFlib.pb" *doc.PbPDF_Doc = PbPDF_New() *page = PbPDF_AddPageA4(*doc) ; Draw a red rectangle PbPDF_Page_SetRGBFill(*page, 1.0, 0.0, 0.0) PbPDF_Page_FillRect(*page, 50, 600, 200, 100) ; Draw a blue circle PbPDF_Page_SetRGBFill(*page, 0.0, 0.0, 1.0) PbPDF_Page_FillCircle(*page, 400, 650, 50) ; Draw a green line PbPDF_Page_SetRGBStroke(*page, 0.0, 0.8, 0.0) PbPDF_Page_SetLineWidth(*page, 3) PbPDF_Page_DrawLine(*page, 50, 500, 500, 500) PbPDF_SaveToFile(*doc, "example2_graphics.pdf") PbPDF_Free(*doc)

Example 3: CJK Text and Encryption

; PbPDFlib - CJK text and encryption XIncludeFile "PbPDFlib.pb" *doc.PbPDF_Doc = PbPDF_New() *page = PbPDF_AddPageA4(*doc) ; Load CJK font *font.PbPDF_Font = PbPDF_LoadTTFontFromFile(*doc, "C:\Windows\Fonts\msyh.ttc", #True) ; Display CJK text PbPDF_Page_ShowTextUTF8Ex(*doc, *page, *font, "Hello, PbPDFlib!", 36) ; Set encryption PbPDF_SetPassword(*doc, "user", "owner", #PbPDF_PERM_PRINT, #PbPDF_ENCRYPT_RC4_128) PbPDF_SaveToFile(*doc, "example3_encrypted.pdf") PbPDF_Free(*doc)
