概述

PbPDFlib 是一个纯 PureBasic 实现的 PDF 文件操作库，无需安装 Adobe Acrobat 或任何第三方依赖，即可创建、读取和修改 PDF 文件。

该库参考 Go 语言的 pdfcpu 项目和 C 语言的 libharu 项目移植而来，以 libharu 的 API 设计风格为主要参考，同时融合 pdfcpu 的读取/修改等高级功能。所有算法均用 PureBasic 原生实现，无第三方依赖。

注意：使用 PbPDFlib 库前，需要在代码开头添加 XIncludeFile "PbPDFlib.pb" 以引入库文件。

主要功能

  • 创建符合 PDF 标准的文档，支持 PDF 1.2-2.0
  • 读取和解析现有 PDF 文件
  • 文本绘制（标准字体 + TrueType 字体 + UTF-8 编码）
  • 图形绘制（路径/描边/填充/颜色/变换矩阵）
  • JPEG 和 PNG 图片加载与绘制
  • TrueType 字体子集化
  • RC4-40/RC4-128/AES-128/AES-256 加密
  • 书签/大纲创建和修改
  • 文本注释、链接注释、高亮注释
  • 文本水印（旋转/透明度/对角线）
  • 页眉/页脚
  • PDF 合并/拆分/提取/删除页面
  • 扩展图形状态（透明度/混合模式）

文档管理

PbPDF_New()
创建一个新的 PDF 文档对象。
返回值： PDF文档指针（PbPDF_Doc）。失败时返回 0。
; 创建一个新PDF文档 *doc.PbPDF_Doc = PbPDF_New() If *doc ; 文档创建成功 EndIf
PbPDF_Free(*doc.PbPDF_Doc)
释放 PDF 文档对象及其所有资源。
参数类型说明
*docPbPDF_DocPDF文档指针
PbPDF_SaveToFile(*doc.PbPDF_Doc, fileName.s)
将 PDF 文档保存到指定文件。
参数类型说明
*docPbPDF_DocPDF文档指针
fileNameString输出文件的完整路径
返回值： 成功返回 #PbPDF_OK，失败返回错误码。
; 保存PDF文件 If PbPDF_SaveToFile(*doc, "output.pdf") = #PbPDF_OK Debug "保存成功！" EndIf
PbPDF_SetPDFVersion(*doc.PbPDF_Doc, version.i)
设置 PDF 版本。支持 PDF 1.2 到 PDF 2.0。
参数类型说明
*docPbPDF_DocPDF文档指针
versionIntegerPDF版本常量（#PbPDF_PDF_VER_12 ~ #PbPDF_PDF_VER_20）
PbPDF_SetCompressionMode(*doc.PbPDF_Doc, mode.i)
设置压缩模式。可组合使用 #PbPDF_COMP_TEXT、#PbPDF_COMP_IMAGE、#PbPDF_COMP_METADATA、#PbPDF_COMP_ALL。
参数类型说明
*docPbPDF_DocPDF文档指针
modeInteger压缩模式标志
PbPDF_GetPageCount(*doc.PbPDF_Doc)
获取文档的页面总数。
返回值： 页面数量（整数）。

示例代码

示例1：创建基本PDF文档

; PbPDFlib - 创建基本PDF文档示例 XIncludeFile "PbPDFlib.pb" ; 创建文档 *doc.PbPDF_Doc = PbPDF_New() ; 添加A4页面 *page = PbPDF_AddPage(*doc) PbPDF_Page_SetPredefinedSize(*page, #PbPDF_PAGE_SIZE_A4, #PbPDF_PAGE_PORTRAIT) ; 写入文本 PbPDF_Page_BeginText(*page) PbPDF_Page_SetFontAndSize(*page, "Helvetica", 24) PbPDF_Page_MoveTextPos(*page, 50, 750) PbPDF_Page_ShowText(*page, "Hello PbPDFlib!") PbPDF_Page_EndText(*page) ; 保存文件 PbPDF_SaveToFile(*doc, "example1.pdf") PbPDF_Free(*doc)

示例2：绘制图形

; PbPDFlib - 绘制图形示例 XIncludeFile "PbPDFlib.pb" *doc.PbPDF_Doc = PbPDF_New() *page = PbPDF_AddPageA4(*doc) ; 绘制红色矩形 PbPDF_Page_SetRGBFill(*page, 1.0, 0.0, 0.0) PbPDF_Page_FillRect(*page, 50, 600, 200, 100) ; 绘制蓝色圆形 PbPDF_Page_SetRGBFill(*page, 0.0, 0.0, 1.0) PbPDF_Page_FillCircle(*page, 400, 650, 50) ; 绘制绿色线条 PbPDF_Page_SetRGBStroke(*page, 0.0, 0.8, 0.0) PbPDF_Page_SetLineWidth(*page, 3) PbPDF_Page_DrawLine(*page, 50, 500, 500, 500) PbPDF_SaveToFile(*doc, "example2_graphics.pdf") PbPDF_Free(*doc)

示例3：中文文本和加密

; PbPDFlib - 中文文本和加密示例 XIncludeFile "PbPDFlib.pb" *doc.PbPDF_Doc = PbPDF_New() *page = PbPDF_AddPageA4(*doc) ; 加载中文字体 *font.PbPDF_Font = PbPDF_LoadTTFontFromFile(*doc, "C:\Windows\Fonts\msyh.ttc", #True) ; 显示中文 PbPDF_Page_ShowTextUTF8Ex(*doc, *page, *font, "你好，PbPDFlib！", 36) ; 设置加密 PbPDF_SetPassword(*doc, "user", "owner", #PbPDF_PERM_PRINT, #PbPDF_ENCRYPT_RC4_128) PbPDF_SaveToFile(*doc, "example3_chinese_encrypted.pdf") PbPDF_Free(*doc)
